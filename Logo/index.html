Social Coding

GitHub es una plataforma de desarrollo colaborativo de software para alojar proyectos utilizando el sistema de control de versiones Git.

Además, puedes contribuir a mejorar el software de los demás. Para poder alcanzar esta meta, GitHub provee de funcionalidades para hacer un fork y solicitar pulls.

StyleCI

Arregle automáticamente el estilo del código de su PHP, Python, JavaScript, JSX, Flow, TypeScript, CSS, SCSS y Less. Habilitar, configurar e ir! Repleto de características con las que no puede vivir, StyleCI proporciona comprobaciones para sus repositorios de código abierto y cerrado. Mediante el análisis de cada solicitud de envío o extracción realizada en su repositorio, StyleCI garantiza que su código siempre se escriba en función de los estándares que desea, con una fijación totalmente automática disponible. ¡También nos integramos con el estado de compromiso de GitHub!
